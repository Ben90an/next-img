Example 1

One size per breakpoint

The most typical expected usage when using this plugin.

<Picture src={require('../images/jelly.jpg?sizes=375,780')} />
  • The image is resized to 1x and 2x density for each specified size
  • This results in 375px, 750px, 780px, 1560px sizes
  • Each of those sizes is converted to webp resulting in 8 images total
  • All images are optimized to the specified quality setting
  • The 375px and 750px images are used on smaller devices up to 768px wide
  • The 780px and 1560px images are used on larger devices above 768px wide
